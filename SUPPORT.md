# Support

This repository is maintained by Peculiar Cloud as a public, open source reference deployment.

## Community Support

Use GitHub Issues for reproducible bugs, documentation gaps, and feature requests. Please include:

- image tag or commit SHA;
- deployment method;
- browser and server logs;
- reverse proxy details, if applicable;
- whether `TLDRAW_LICENSE_KEY` is configured.

Do not post private room contents, uploaded assets, secrets, or license keys.
